# Subtitle File Naming Convention

**Inherited from Hub-level rule. Always keep original filenames, append language code.**

## Rule

1. **ALWAYS keep the original filename** as the base.
2. **Append the language code** (`_en`, `_vi`, `_ja`, etc.) before the extension.
3. **NEVER rename** to generic names like `subtitle_en.srt`.

## Examples

| Original | ✅ Correct | ❌ Wrong |
|---|---|---|
| `my_video.srt` | `my_video_en.srt` | `subtitle_en.srt` |
| `episode_01.ass` | `episode_01_vi.ass` | `translated.ass` |
