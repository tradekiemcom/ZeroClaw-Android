# 🌀 ZeroClaw-Android: Omni-Agent Dashboard

Dự án cài đặt **[ZeroClaw Core](https://github.com/zeroclaw-labs/zeroclaw)** chuyên biệt để biến thiết bị **Android** thành một Siêu Trợ Lý Đa Năng. Toàn bộ việc quản lý được thực hiện trực quan thông qua Giao diện Web (GUI).

---

## 🌟 Chức Năng Cốt Lõi
- **Dashboard-First**: Quản lý Skill, Tool và Task 100% qua trình duyệt.
- **Omni-Agent v16.1**: Một Agent duy nhất xử lý đa tác vụ (Trading, News, Planning).
- **Core Automation**: Tích hợp sẵn các mẫu kỹ năng phân tích và lấy tin tức thực chiến.

## 🚀 Cài Đặt Nhanh (trên thiết bị Android > Termux)
```bash
pkg update -y && pkg install -y git
git clone https://github.com/tradekiemcom/ZeroClaw-Android.git
cd ZeroClaw-Android
chmod +x install.sh && ./install.sh
```
